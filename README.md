# DigiTools

A digital tools buying website where you can browse and buy premium digital products like AI tools, design assets, and templates.

## Built With
- React.js
- Tailwind CSS
- DaisyUI
- React-Toastify

## Features
- Browse 6 premium digital products and add them to cart
- Cart shows selected items with total price and remove option
- Toast notifications for add to cart, remove, and checkout actions

# DigiTools

DigiTools is a modern e-commerce platform for digital products where users can explore and purchase premium tools such as AI software, design assets, and ready-made templates.

## Built With
- React.js
- Tailwind CSS
- DaisyUI
- React Toastify


## Features
- Browse a curated collection of 6 premium digital products
- Add items to cart with real-time updates
- View cart with selected items and total price calculation
- Remove items from cart easily
- Smooth toast notifications for add, remove, and checkout actions
