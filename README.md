
# DigiTools

DigiTools is a modern e-commerce platform for digital products where users can explore and purchase premium tools such as AI software, design assets, and ready-made templates.

## Built With
React.js
Tailwind CSS
DaisyUI
React Toastify


## Features
Browse a curated collection of 6 premium digital products
Add items to cart with real-time updates
View cart with selected items and total price calculation
Remove items from cart easily
Smooth toast notifications for add, remove, and checkout actions
